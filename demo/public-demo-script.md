# Roteiro 2 min

## Frase central da demo
Não é um prompt pack. É um sistema operacional de trabalho para criadores não-devs usarem Claude com contexto, workflows, decisões e entregas consistentes.

## Sequência prática

1.  Abrir repo no Claude Code
2.  Rodar claude-code/CMD_HANDS_OFF_PUBLIC_DEMO.md
3.  Revisar /demo
4.  Gravar tela seguindo screen-recording-shotlist.md
5.  Subir vídeo no LinkedIn, YouTube não listado ou Loom
6.  Colocar link no README
7.  Publicar GitHub
8.  Publicar Medium
9.  Publicar LinkedIn com CTA para beta feedback

## Não faça na primeira demo

| Evitar | Motivo |
|---|---|
| Explicar RAG primeiro | ICP não-dev pode não entender |
| Mostrar muitos arquivos | Parece complexo |
| Mostrar terminal | Pode assustar público inicial |
| Mostrar dados privados | Risco reputacional |
| Falar “skill family” demais | Parece técnico, não produto |
| Fazer vídeo longo | Baixa retenção |
