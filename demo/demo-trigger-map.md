# Trigger Map para Demo Pública

## Propósito

Mapear os triggers que ativam os workflows do Creator Operating Suite, demonstrando a facilidade de uso para criadores não-técnicos.

## Triggers Primários (Exemplos para a Demo)

| Trigger | Intenção do Usuário | Skill Ativada (Exemplo) |
|---|---|---|
| `/cmd-hands-off-public-demo` | Iniciar um workflow de criação de conteúdo | `creator-entry-skill` + `personalization-os` |
| `/create-linkedin-post` | Gerar um post para LinkedIn | `visual-canvas-forge` (para imagem) ou `onboarding-xray-ebook` (para texto) |
| `/analyze-audience` | Analisar público-alvo | `research-orchestrator-cmd` |

## Como Funciona (Visão Simplificada)

1.  **Comando Simples:** O usuário digita um comando intuitivo no Claude.
2.  **Roteamento Inteligente:** O sistema interpreta a intenção e roteia para a skill mais adequada.
3.  **Contexto Automático:** O contexto do usuário (persona, tom, estilo) é automaticamente carregado.
4.  **Output Estruturado:** A skill executa e entrega um resultado consistente e pronto para uso.

## Benefícios para o Criador

*   **Redução de Esforço:** Não precisa reexplicar o contexto.
*   **Consistência:** Outputs padronizados e de alta qualidade.
*   **Velocidade:** Geração rápida de conteúdo e artefatos.
*   **Organização:** Tudo é logado e rastreável.
