# Checklist de Release para GitHub

## Antes da Publicação

- [x] Gravar vídeo de até 2 min (T-001)
- [ ] Adicionar link do vídeo no README (T-002)
- [ ] Criar 1 GIF curto de 10–20s (T-003) - *Opcional*
- [ ] Criar beta feedback form (T-004) - *Recomendado*
- [ ] Adicionar screenshots em `/assets/screenshots` (T-005) - *Recomendado*
- [x] Publicar Medium post (T-006) - *Já gerado*

## Após a Publicação

- [ ] Post LinkedIn com vídeo (T-007)
- [ ] Medir sinais (Stars, DMs, comentários, beta requests) (T-008)

## Sequência Prática

1.  Abrir repo no Claude Code
2.  Rodar `claude-code/CMD_HANDS_OFF_PUBLIC_DEMO.md`
3.  Revisar `/demo`
4.  Gravar tela seguindo `screen-recording-shotlist.md`
5.  Subir vídeo no LinkedIn, YouTube não listado ou Loom
6.  Colocar link no README
7.  Publicar GitHub
8.  Publicar Medium
9.  Publicar LinkedIn com CTA para beta feedback
